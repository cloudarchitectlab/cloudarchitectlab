# Cloud Architect Lab Brand Guide

Cloud Architect Lab is a professional cloud architecture portfolio focused on practical infrastructure, architecture documentation, and case-study based learning.

## Brand Position

Cloud Architect Lab represents the technical portfolio work of an emerging cloud and AI solutions architect. The brand should feel clear, credible, technical, and enterprise-ready.

## Tagline

Infrastructure as Code. Architecture as Code.

## Core Themes

- Cloud architecture
- Infrastructure as Code
- Architecture as Code
- Enterprise documentation
- Secure, repeatable lab environments
- Practical case studies
- Privacy-preserving portfolio work

## Privacy Standard

Cloud Architect Lab does not publish employer names, customer names, production infrastructure details, account IDs, secrets, private credentials, live endpoints, or sensitive organizational information.

Use general language such as:

- enterprise technology
- regulated environments
- mission-critical systems
- sensitive data environments
- compliance-driven organizations
- cloud architecture labs

## Logo Usage

Use the CAL logo or CAL monogram consistently across:

- Cloud Architect Lab website
- GitHub profile
- GitHub repositories
- LinkedIn banner
- LinkedIn Projects
- Architecture diagrams
- README files
- Documentation

Avoid stretching, recoloring, or distorting the logo.

## Documentation Standards

Every published case study should include:

- Business Scenario
- Requirements
- Architecture Diagram
- Design Decisions
- Terraform / IaC
- Deployment Steps
- Validation
- Lessons Learned
- Cleanup
- Future Enhancements

## Publishing Workflow

Every completed project should produce:

1. GitHub Repository
2. GitHub README
3. Cloud Architect Lab website article
4. LinkedIn Project entry
5. LinkedIn post
6. Architecture diagrams
7. Supporting documentation

## Long-Term Goal

Cloud Architect Lab exists to demonstrate practical cloud architecture, infrastructure engineering, and AI solution design through documented, repeatable case studies rather than isolated labs.

## Brand Rule

**The brand supports the work. The work is the portfolio.**
